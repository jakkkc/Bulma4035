CSS library (Bulma)

Bulma is a free and open-source CSS framework based on Flexbox, used to quickly create responsive and mobile-friendly websites. It is written in Sass and is highly customizable, allowing developers to easily modify the default styles to match their design needs. Bulma is also lightweight, with minimal CSS and no JavaScript, making it a popular choice for building simple and clean websites.

REACT.js

React.js is a JavaScript library for building user interfaces. It was developed by Facebook and is now maintained by a large community of developers. React allows developers to build reusable UI components and manage the state of their application in an efficient and predictable way.

It is the core framework to this course

Material UI

We will also learn this Amazing REACT UI.

Material-UI is a popular front-end library that implements the Material Design guidelines developed by Google. It is built with React and provides a set of pre-built UI components, such as buttons, forms, and navigation elements, that can be easily integrated into a React application. These components are designed to look and function consistently across different platforms and devices, and can be customized to match the look and feel of your application. Material-UI also includes features such as theming, which allows developers to quickly change the overall visual style of the components, and accessibility, to support users with disabilities.
